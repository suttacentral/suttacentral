Varuṇavagga

4. Rukkha­dhamma­jātaka

“Sādhū sambahulā ñātī,
api rukkhā araññajā;
Vāto vahati ekaṭṭhaṃ,
brahantampi vanappatin”ti.

Rukkha­dhamma­jātakaṃ catutthaṃ.
