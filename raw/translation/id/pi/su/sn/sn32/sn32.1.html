Saṃyutta Nikāya

32. Kelompok Khotbah tentang Deva Awan

1. Versi Sederhana

Di Sāvatthī. “Para bhikkhu, Aku akan mengajarkan kepada kalian mengenai para deva dari kelompok yang menetap di awan. Dengarkanlah …

“Dan apakah, para bhikkhu, para deva dari kelompok yang menetap di awan? Ada, para bhikkhu, para deva awan-dingin, para deva awan-hangat, para deva awan-badai, para deva awan-angin, dan para deva awan-hujan.

“Ini, para bhikkhu, disebut para deva dari kelompok yang menetap di awan.”
